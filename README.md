# ASL-tensorflow
This is a repository that has my training/implementation of the ASL dataset using tensorflow

- This has been tested using
  - Ubuntu 18.04.6
  - Python 3.9

---
## How to start
Start Project by downloading the requirements.txt

`cd ASL-tensorflow/`

`pip install -r requirements.txt`
